About grpc-demo
====================
>This is a test project


Prerequisites
=============
* Java 1.8

Run
=============
* 1.You need install maven.
* 2.config pom.xml
```
Edit hello.proto
add pom.xml plugin

./mvn protobuf:compile
./mvn protobuf:compile-custom
Copy HelloGrpc and HelloOuterClass into correct package.
```
* 3.Run HelloServer.java
* 4.Run HelloClient.java


Building 
========
./mvn clean build
